# FaceRecognition
Library project for face detection and recognition using Machine Learning with CNN model


Installation 
The library openCV2, tensorflow and numpy are needed for running this project.  

I advice to install a virtual environnement by using :
- python3 -m pip install virtualenv 

Linux 
- sudo apt update 
- sudo apt install python3.6
- python3 -m pip install opencv-python 
- python3 -m pip install tensorflow

MacOS
- brew install python3.6
- python3 -m pip install opencv-python 
- python3 -m pip install tensorflow


Initialising a user

Create the following path to put the images of your users in folders with the names's user: 
- /Data/Raw_img/ 

Then run, go through the main.py script and run: 
- face_recognition.user_init(rawimg, "User_Name")


Utilisation 
Exemple of methods using are provided in the main.py script for : training, predicting, saving...
